# Ganesh-
उपयोगकर्ता अपनी जन्म तिथि, समय और स्थान दर्ज करके व्यक्तिगत कुंडली और राशिफल प्राप्त कर सकते हैं।   वेबसाइट पर दैनिक, साप्ताहिक और मासिक राशिफल उपलब्ध हैं। प्रश्नों का उत्तर देता है और सुझाव प्रदान करता है। साथ ही ब्लॉग सेक्शन में ज्योतिष से जुड़ी जानकारी, उपाय और मंत्र दिए गए हैं। उपयोगकर्ता चाहें तो कंसल्टेशन और बुकिंग भी कर सकते हैं।
